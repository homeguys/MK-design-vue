# 简介

汇集了一些常用的工具函数，特别是在业务中特定的功能，旨在简化日常开发任务，提高代码效率。

## 安装

使用npm或yarn安装

```
npm install @MK/utils
```

```
yarn install @MK/utils
```

## 🌟 核心特性

- **模块化设计**：支持按需引入，Tree-Shaking 友好
- **类型完备**：完整TypeScript支持
- **浏览器支持**：现代浏览器支持好
